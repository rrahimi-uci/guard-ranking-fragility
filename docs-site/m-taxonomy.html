Hazard taxonomy

The hazard taxonomy the guards label against — the safe/unsafe decision, hazard categories, and how benchmark labels map onto them (positive class = unsafe).

One canonical label space (agent_bouncer.core.taxonomy.Hazard), aligned to the MLCommons AILuminate hazard set and extended with two agent-specific surfaces.
